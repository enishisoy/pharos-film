# PHAROS FILM

静的HTML / CSS / JavaScriptで構成したPHAROS FILMのブランドサイトです。

## 構成

- `index.html`：ページ本体
- `style.css`：デザイン・レスポンシブ
- `script.js`：メニュー・スクロール演出
- `images/`：PowerPointから抽出した画像

## ローカル確認

`index.html`をブラウザで開いてください。

より安定して確認する場合は、VS Code拡張機能「Live Server」を使えます。

## 問い合わせフォーム

現在のフォーム送信先は仮です。

`index.html`内の次の部分を、Formspreeなどで取得したURLへ変更してください。

```html
<form action="https://formspree.io/f/your-form-id" method="POST">
```
